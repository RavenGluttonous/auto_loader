AutoLoader 使用说明
=====================

一、系统要求
-----------
1. Windows 10/11 操作系统
2. PostgreSQL 数据库（建议版本 14.x）
3. 扫描器设备（如有使用）

二、安装步骤
-----------
1. 解压部署包到任意位置（如桌面或程序文件夹）
2. 确保PostgreSQL数据库已安装并正确配置
3. 运行 setup_config.bat 进行初始配置
4. 编辑 config.py 文件，设置数据库连接参数
5. 双击 auto_loader.exe 运行程序

三、配置文件说明
-------------
配置文件(config.py)包含以下重要参数：

1. 数据库参数:
   - DB_HOST: 数据库服务器地址（默认localhost）
   - DB_PORT: 数据库服务器端口（默认5432）
   - DB_NAME: 数据库名称
   - DB_USER: 数据库用户名
   - DB_PASSWORD: 数据库密码

2. 设备参数:
   - SCANNER_PORT: 扫描器设备COM端口（如COM3）

四、常见问题
-----------
1. 程序无法启动
   - 检查配置文件是否正确
   - 确保数据库服务已启动
   - 查看logs文件夹下的日志文件获取详细错误信息

2. 无法连接数据库
   - 检查数据库连接参数
   - 确认PostgreSQL服务已启动
   - 验证防火墙设置是否允许数据库连接

3. 扫描器无法使用
   - 检查设备是否正确连接
   - 验证COM端口设置是否正确
   - 确认设备驱动已安装

五、联系方式
-----------
如遇到问题，请联系系统管理员或开发人员获取帮助。
